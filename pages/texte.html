Les balises de texte en HTML

🏠 Accueil

Introduction

Dans cette section, vous allez découvrir les balises utilisées pour afficher et styliser le **texte dans HTML**. Les balises de texte permettent de structurer le contenu visible de la page web.

1️⃣ Les balises de titre

Les balises de titre permettent de structurer la hiérarchie de la page. Il existe 6 niveaux de titres, de <h1> (le plus important) à <h6> (le moins important).

Exemple :

                    
<h1>Titre de niveau 1</h1>
<h2>Titre de niveau 2</h2>
<h3>Titre de niveau 3</h3>
<h4>Titre de niveau 4</h4>
<h5>Titre de niveau 5</h5>
<h6>Titre de niveau 6</h6>

👉 **Bonnes pratiques** :

  • Utilisez **une seule balise `<h1>`** par page, car c'est le titre principal de la page.
  • Utilisez les autres titres (**`<h2>` à `<h6>`**) pour structurer vos sections de contenu.

2️⃣ La balise de paragraphe

Pour afficher du texte normal, on utilise la balise **`<p>`**. Chaque paragraphe de texte doit être contenu dans une balise `

`.

Exemple :

                    
<p>Ceci est un paragraphe de texte.</p>
<p>Et ceci est un autre paragraphe.</p>

👉 **Bonnes pratiques** :

  • Utilisez la balise **`<p>`** pour structurer les paragraphes.
  • Ne mettez pas tout le texte dans une seule balise **`<p>`**, divisez-le en plusieurs paragraphes.

3️⃣ Les balises de mise en forme

Ces balises permettent de mettre en forme le texte, comme le rendre **gras**, **italique**, ou **souligné**.

Les principales balises :

  • <strong> : Met le texte en **gras** (important sémantiquement).
  • <em> : Met le texte en **italique** (pour insister sur un mot).
  • <b> : Met le texte en **gras** (mais sans importance sémantique).
  • <i> : Met le texte en **italique** (sans importance sémantique).
  • <u> : **Souligne** le texte (rarement utilisé).

Exemple :

                    
<p>Texte en <strong>gras</strong>.</p>
<p>Texte en <em>italique</em>.</p>
<p>Texte en <b>gras (non sémantique)</b>.</p>
<p>Texte en <u>souligné</u>.</p>

4️⃣ Les citations et les citations longues

La balise <blockquote>

Utilisez la balise **`<blockquote>`** pour citer des paragraphes entiers.

                    
<blockquote>
Le HTML est le langage de base pour créer des pages Web. 
Il est utilisé pour structurer le contenu.
</blockquote>

La balise <q>

Utilisez la balise **`<q>`** pour citer une petite phrase au sein d'un paragraphe.

                    
<p>Le professeur a dit : <q>N'abandonnez jamais !</q></p>

5️⃣ Exercice pratique

Créez une nouvelle page **texte.html** avec les consignes suivantes :

  • 1️⃣ Ajoutez un titre principal en **`<h1>`** : "Ma première page de texte".
  • 2️⃣ Ajoutez trois **paragraphes** avec des informations personnelles (loisirs, travail, etc.).
  • 3️⃣ Mettez un mot en **gras** et un autre en **italique** dans chaque paragraphe.
  • 4️⃣ Ajoutez une **citation** avec **`<blockquote>`** à la fin de la page.

➡️ Ouvrez le fichier dans le navigateur et vérifiez le résultat !

6️⃣ Points à retenir

  • 📌 Les **titres** vont de `<h1>` (le plus important) à `<h6>` (le moins important).
  • 📌 Les **paragraphes** sont créés avec la balise **`<p>`**.
  • 📌 Les **mises en forme** se font avec `<strong>`, `<em>`, `<b>`, `<i>`, et `<u>`.
  • 📌 Les **citations** utilisent **`<blockquote>`** pour des longs paragraphes et **`<q>`** pour les petites citations.
