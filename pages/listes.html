Les listes en HTML

🏠 Accueil

Introduction

Les **listes HTML** permettent de regrouper des éléments de manière organisée. Il existe 3 types de listes principales en HTML :

  • 🔹 **Listes non ordonnées** (balise <ul>)
  • 🔹 **Listes ordonnées** (balise <ol>)
  • 🔹 **Listes de définition** (balise <dl>)

1️⃣ Les listes non ordonnées

Les **listes non ordonnées** affichent des éléments sous forme de **puces**. Chaque élément de la liste est contenu dans une balise **`<li>`** (élément de liste).

Exemple :

                    <ul>
                        <li>HTML</li>
                        <li>CSS</li>
                        <li>JavaScript</li>
                    </ul>

👉 Ce code affiche :

  • HTML
  • CSS
  • JavaScript

👉 **Bonnes pratiques** :

  • Utilisez les listes **non ordonnées** pour les contenus **sans ordre particulier**.
  • Chaque élément de liste doit être dans une balise **`<li>`**.

2️⃣ Les listes ordonnées

Les **listes ordonnées** affichent des éléments avec des **numéros** ou des **lettres**. Chaque élément est contenu dans une balise **`<li>`** (élément de liste).

Exemple :

                    <ol>
                        <li>Préparer les ingrédients</li>
                        <li>Cuire au four pendant 20 min</li>
                        <li>Laisser refroidir</li>
                    </ol>

👉 Ce code affiche :

  1. Préparer les ingrédients
  2. Cuire au four pendant 20 min
  3. Laisser refroidir

👉 **Bonnes pratiques** :

  • Utilisez les listes **ordonnées** pour les étapes ou **instructions chronologiques**.
  • Chaque élément de liste doit être dans une balise **`<li>`**.

3️⃣ Les listes imbriquées

Les **listes imbriquées** permettent de créer des sous-listes à l'intérieur d'une liste principale.

Exemple :

                    <ul>
                    <li>HTML</li>
                    <li>CSS
                        <ul>
                            <li>Flexbox</li>
                            <li>Grid</li>
                        </ul>
                    </li>
                    <li>JavaScript</li>
                    </ul>

👉 Ce code affiche :

  • HTML
  • CSS
    • Flexbox
    • Grid
  • JavaScript

4️⃣ Les listes de définition

Les **listes de définition** sont utilisées pour définir des termes, des concepts ou des définitions. Elles utilisent 3 balises :

  • 🔹 **`<dl>`** : Définit la liste de définition.
  • 🔹 **`<dt>`** : Définit le terme à expliquer.
  • 🔹 **`<dd>`** : Définit la description du terme.

Exemple :

                    <dl>
                        <dt>HTML</dt>
                        <dd>HyperText Markup Language</dd>

                        <dt>CSS</dt>
                        <dd>Cascading Style Sheets</dd>

                        <dt>JavaScript</dt>
                        <dd>Langage de programmation pour le web</dd>
                    </dl>

👉 Ce code affiche :

HTML
HyperText Markup Language
CSS
Cascading Style Sheets
JavaScript
Langage de programmation pour le web

6️⃣ Points à retenir

  • 📌 Utilisez **`<ul>`** pour les **listes non ordonnées** (avec des puces).
  • 📌 Utilisez **`<ol>`** pour les **listes ordonnées** (avec des numéros ou lettres).
  • 📌 Les **listes imbriquées** permettent de créer des sous-listes.
  • 📌 Les **listes de définition** utilisent **`<dl>`**, **`<dt>`** et **`<dd>`**.
