La structure de base d'une page HTML

🏠 Accueil

Introduction

Dans ce chapitre, nous allons apprendre à structurer correctement une page HTML. Vous découvrirez les balises essentielles et leur rôle dans la construction d'une page Web.

1️⃣ Structure de base d'une page HTML

Voici le squelette de base d'une page HTML :

                
<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Titre de la page</title>
</head>
<body>
    <h1>Bienvenue sur ma page</h1>
</body>
</html>

👉 Ce squelette est **obligatoire** pour que le navigateur comprenne qu'il s'agit d'un document HTML5.

2️⃣ Explications des balises essentielles

1. La balise <!DOCTYPE html>

Cette balise annonce au navigateur qu'il s'agit d'un document HTML5. Cette déclaration doit être **au tout début** du fichier HTML.

2. La balise <html>

La balise <html> est la balise racine qui **englobe tout le contenu** de la page. L'attribut lang="fr" indique que le contenu de la page est en français, ce qui aide au référencement SEO et à l'accessibilité.

3. La balise <head>

Le **<head>** contient les **informations invisibles** de la page, comme le titre, la langue et les liens vers des ressources externes (CSS, polices, etc.).

  • <meta charset="UTF-8"> : Indique au navigateur d'utiliser le format UTF-8 (pour les accents et caractères spéciaux).
  • <title> : Définit le **titre de la page** (visible dans l'onglet du navigateur).
  • <link> : Permet de lier des fichiers CSS.

4. La balise <body>

La balise **<body>** contient **tout le contenu visible** de la page : titres, paragraphes, images, vidéos, etc. Ce que l'utilisateur voit à l'écran se trouve toujours dans le **<body>**.

3️⃣ Bonnes pratiques

  • 🔹 Toujours commencer avec <!DOCTYPE html> pour indiquer au navigateur qu'il s'agit d'un document HTML5.
  • 🔹 Ajoutez toujours **lang="fr"** dans la balise <html> pour indiquer la langue du contenu.
  • 🔹 Mettez le titre de la page dans la balise <title> car il apparaît dans l'onglet du navigateur.
  • 🔹 Évitez de mélanger du contenu **visuel** dans la balise **<head>** (placez tout dans le **<body>**).

4️⃣ Exercice pratique

Essayez de créer votre propre page HTML en respectant les consignes suivantes :

  • 1️⃣ Créez un fichier **page.html**.
  • 2️⃣ Ajoutez le **squelette de base** que nous avons vu plus haut.
  • 3️⃣ Ajoutez un titre de page personnalisé dans la balise **<title>**.
  • 4️⃣ Ajoutez un titre principal dans le **<body>** (par exemple, "Ma première page HTML").

➡️ Une fois le fichier prêt, ouvrez-le dans votre navigateur et vérifiez le résultat !

5️⃣ Points à retenir

  • 📌 La balise <!DOCTYPE html> est obligatoire en début de fichier.
  • 📌 La balise <html> est la racine du document.
  • 📌 La balise <head> contient les métadonnées, et la balise <body> contient le contenu visible.
