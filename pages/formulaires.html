Les formulaires en HTML

🏠 Accueil

Introduction

Les **formulaires HTML** permettent de collecter des informations auprès des utilisateurs. Les données saisies peuvent être envoyées à un serveur pour être traitées. Dans ce chapitre, vous apprendrez à créer et personnaliser un formulaire HTML.

1️⃣ Structure de base d'un formulaire

Un formulaire HTML est défini par la balise **`<form>`**. Voici sa structure de base :

<form action="url_du_serveur" method="post">
  <!-- Champs de formulaire ici -->
  <button type="submit">Envoyer</button>
</form>
  • 🔹 **`action`** : URL vers laquelle les données du formulaire sont envoyées.
  • 🔹 **`method="post"`** ou **`method="get"`** : Méthode de soumission (POST ou GET).

2️⃣ Les champs de formulaire

1. Champ de texte (`<input type="text">`)

<label for="nom">Nom :</label>
<input type="text" id="nom" name="nom" placeholder="Entrez votre nom">

2. Champ de mot de passe (`<input type="password">`)

<label for="motdepasse">Mot de passe :</label>
<input type="password" id="motdepasse" name="motdepasse">

3. Champ de saisie d'e-mail (`<input type="email">`)

<label for="email">Email :</label>
<input type="email" id="email" name="email" placeholder="Entrez votre email">

4. Zone de texte (`<textarea>`)

<label for="message">Message :</label>
<textarea id="message" name="message" rows="4" cols="50"></textarea>

5. Bouton de soumission (`<button>`)

<button type="submit">Envoyer</button>

3️⃣ Autres champs de formulaire

1. Boutons radio (`<input type="radio">`)

<input type="radio" name="genre" value="homme"> Homme
<input type="radio" name="genre" value="femme"> Femme
Homme Femme

2. Cases à cocher (`<input type="checkbox">`)

<input type="checkbox" name="abonnement" value="newsletter"> S'abonner à la newsletter
S'abonner à la newsletter

3. Listes déroulantes (`<select>`)

<select name="pays">
  <option value="france">France</option>
  <option value="espagne">Espagne</option>
  <option value="italie">Italie</option>
</select>

4️⃣ Exercice pratique

Créez une nouvelle page **formulaires.html** avec les consignes suivantes :

  • 1️⃣ Créez un formulaire complet avec les champs suivants :
  • 🔹 **Nom** (input texte)
  • 🔹 **Email** (input email)
  • 🔹 **Mot de passe** (input password)
  • 🔹 **Genre** (boutons radio : Homme, Femme, Autre)
  • 🔹 **Choix du pays** (liste déroulante avec 3 pays)
  • 🔹 **Zone de message** (textarea)
  • 🔹 **Case à cocher** pour accepter les **conditions d'utilisation**.

➡️ Ouvrez la page dans le navigateur et vérifiez le résultat !

5️⃣ Points à retenir

  • 📌 La balise **`<form>`** permet de créer un formulaire.
  • 📌 Les champs de saisie incluent **`<input>`**, **`<textarea>`**, **`<select>`**, etc.
  • 📌 N'oubliez pas d'ajouter des **labels** pour chaque champ de saisie.
  • 📌 Les types d'input incluent **text**, **password**, **email**, **radio**, **checkbox**, etc.
  • 📌 Les formulaires peuvent utiliser les méthodes **GET** et **POST**.

6️⃣ Exemples de formulaires

Voici quelques exemples de formulaires couramment utilisés sur les sites web :

  • 🔹 **Formulaire de contact**
  • 🔹 **Formulaire de connexion**
  • 🔹 **Formulaire d'inscription**
  • 🔹 **Formulaire de recherche**
  • 🔹 **Formulaire de paiement**

Formulaire de contact

Ce formulaire est utilisé pour permettre aux visiteurs de **vous contacter**. Il inclut généralement les champs **Nom**, **Email**, **Message**, et un **bouton Envoyer**.

Formulaire de connexion

Le **formulaire de connexion** est utilisé pour permettre aux utilisateurs de se connecter à leur compte. Il inclut **Email**, **Mot de passe** et un **bouton de connexion**.

Formulaire d'inscription

Le **formulaire d'inscription** est utilisé pour enregistrer de nouveaux utilisateurs sur le site. Il inclut les champs **Nom**, **Email**, **Mot de passe**, **Confirmation du mot de passe** et un **bouton d'inscription**.

Formulaire de recherche

Le **formulaire de recherche** est utilisé pour rechercher des informations sur un site. Ce formulaire est souvent simple et comprend un **champ de recherche** et un **bouton de recherche**.

Formulaire de paiement

Le **formulaire de paiement** permet de collecter les **informations de paiement**. Il inclut des champs pour le **nom**, le **numéro de carte de crédit**, la **date d'expiration** et le **CVC**.

Points à retenir

  • 📌 **Formulaire de contact** : Permet à l'utilisateur de vous envoyer un message.
  • 📌 **Formulaire de connexion** : Permet aux utilisateurs de se connecter à un compte.
  • 📌 **Formulaire d'inscription** : Permet aux utilisateurs de créer un nouveau compte.
  • 📌 **Formulaire de recherche** : Permet de rechercher des informations sur le site.
  • 📌 **Formulaire de paiement** : Permet de collecter des informations de carte bancaire.
