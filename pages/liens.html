Les liens en HTML

🏠 Accueil

Introduction

Les **liens hypertextes** sont au cœur du web. Ils permettent de relier des pages entre elles ou de pointer vers d'autres ressources (fichiers, vidéos, etc.). La balise HTML utilisée pour créer un lien est **``**.

1️⃣ Structure de la balise de lien

La balise **`<a>`** (pour "anchor", ancre en anglais) permet de créer un lien cliquable. Elle utilise l'attribut **`href`** pour indiquer l'URL de la page ou de la ressource à atteindre.

Exemple simple

                    <a href="https://www.google.com">Visiter Google</a>

👉 Ce lien envoie l'utilisateur sur **Google** en cliquant dessus.

Structure complète

Voici la structure d'une balise de lien complète :

                    <a href="URL" target="_blank">Texte du lien</a>
  • 🔹 **`href="URL"`** : Lien vers la ressource (autre page, document, etc.).
  • 🔹 **`target="_blank"`** : Ouvre le lien dans un **nouvel onglet**.

2️⃣ Liens internes et externes

Liens internes

Un **lien interne** pointe vers une autre page du **même site web**.

                    <a href="contact.html">Page de contact</a>

👉 Ce lien envoie l'utilisateur sur la page **contact.html** qui se trouve dans le même répertoire que la page actuelle.

Liens externes

Un **lien externe** pointe vers une ressource extérieure au site, comme un autre site web.

                    <a href="https://www.wikipedia.org" target="_blank">Visiter Wikipedia</a>

👉 Ce lien ouvre **Wikipedia** dans un **nouvel onglet** grâce à l'attribut **target="_blank"**.

3️⃣ Liens d'ancrage (navigation vers une section de la même page)

Un **lien d'ancrage** permet de faire défiler la page vers une **section spécifique**. On utilise un **ID** sur l'élément cible et un lien de ce type :

                <a href="#section2">Aller à la section 2</a>

Pour que ce lien fonctionne, la section cible doit avoir un **ID** correspondant :

                <section id="section2">
                <h2>Section 2</h2>
                </section>

👉 En cliquant sur le lien, la page défilera automatiquement jusqu'à la section portant l'ID **section2**.

4️⃣ Autres types de liens

Liens vers un e-mail (mailto:)

Il est possible de créer des liens qui ouvrent un logiciel de messagerie pour envoyer un e-mail.

                    <a href="mailto:contact@example.com">Envoyer un email</a>

👉 En cliquant sur ce lien, le client de messagerie par défaut s'ouvre avec l'adresse e-mail préremplie.

Liens vers un fichier à télécharger

Pour forcer le **téléchargement d'un fichier**, on utilise l'attribut **download** :

                    <a href="fichiers/monfichier.pdf" download>Télécharger le fichier PDF</a>

👉 En cliquant sur ce lien, le fichier PDF est téléchargé au lieu d'être affiché dans le navigateur.

5️⃣ Bonnes pratiques

  • 🔹 Utilisez **des descriptions claires** dans le texte du lien (évitez "cliquez ici").
  • 🔹 Ajoutez **target="_blank"** pour les **liens externes**.
  • 🔹 Évitez d'utiliser des **liens cassés** (vérifiez vos URL).

6️⃣ Exercice pratique

Créez une nouvelle page **liens.html** avec les consignes suivantes :

  • 1️⃣ Ajoutez un **lien interne** vers une page **contact.html**.
  • 2️⃣ Ajoutez un **lien externe** vers **Google** qui s'ouvre dans un nouvel onglet.
  • 3️⃣ Ajoutez un **lien d'ancrage** vers une section de la page portant l'ID **#about**.
  • 4️⃣ Ajoutez un **lien vers un e-mail** (par exemple : contact@example.com).

➡️ Ouvrez la page dans le navigateur et testez les liens.

7️⃣ Points à retenir

  • 📌 La balise **`<a>`** est utilisée pour créer des **liens cliquables**.
  • 📌 Utilisez **`href`** pour spécifier l'URL et **`target="_blank"`** pour ouvrir un nouvel onglet.
  • 📌 Les **liens internes** pointent vers des pages de votre propre site.
  • 📌 Les **liens d'ancrage** permettent de naviguer vers une section de la page.
