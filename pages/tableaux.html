Les tableaux en HTML

🏠 Accueil

Introduction

Les **tableaux HTML** permettent d'organiser des **données sous forme de grille** (lignes et colonnes). Ils sont très utilisés pour afficher des **informations tabulaires** (listes de produits, résultats de tests, etc.).

1️⃣ Structure de base d'un tableau

Un tableau HTML est défini avec la balise **`<table>`**. Il est composé de **lignes** (**`<tr>`**) et de **cellules** (**`<td>`** ou **`<th>`**).

Exemple de tableau simple

                    <table>
                    <tr>
                        <th>Nom</th>
                        <th>Âge</th>
                        <th>Ville</th>
                    </tr>
                    <tr>
                        <td>Alice</td>
                        <td>25</td>
                        <td>Paris</td>
                    </tr>
                    <tr>
                        <td>Bob</td>
                        <td>30</td>
                        <td>Lyon</td>
                    </tr>
                    </table>

👉 **Résultat** :

Nom Âge Ville
Alice 25 Paris
Bob 30 Lyon

2️⃣ Les balises essentielles des tableaux

1. Balise <table>

Cette balise contient **tout le tableau**.

2. Balise <tr>

La balise **`<tr>`** représente **une ligne du tableau**.

3. Balise <th>

La balise **`<th>`** représente **un en-tête de colonne**. Le texte est **gras et centré par défaut**.

4. Balise <td>

La balise **`<td>`** représente **une cellule**. Elle contient les **données du tableau**.

3️⃣ Sections avancées : <thead>, <tbody> et <tfoot>

1. Balise <thead>

Cette balise contient les **en-têtes de colonnes** (les titres de chaque colonne).

2. Balise <tbody>

Cette balise contient le **corps du tableau** (les données du tableau).

3. Balise <tfoot>

Cette balise contient **le pied de tableau** (par exemple, les totaux ou les notes de bas de page). 

                    
                    <table>
                    <thead>
                        <tr>
                        <th>Produit</th>
                        <th>Prix</th>
                        </tr>
                    </thead>
                    <tbody>
                        <tr>
                        <td>Chaise</td>
                        <td>50€</td>
                        </tr>
                        <tr>
                        <td>Table</td>
                        <td>100€</td>
                        </tr>
                    </tbody>
                    <tfoot>
                        <tr>
                        <td>Total</td>
                        <td>150€</td>
                        </tr>
                    </tfoot>
                    </table>

4️⃣ Exercice pratique

Créez une nouvelle page **tableaux.html** avec les consignes suivantes :

  • 1️⃣ Créez un tableau pour afficher une liste de **5 employés** avec les colonnes : **Nom**, **Poste**, **Âge**, **Ville**.
  • 2️⃣ Ajoutez un **thead**, **tbody** et **tfoot**.
  • 3️⃣ Dans le **tfoot**, affichez le nombre total d'employés.

➡️ Ouvrez la page dans le navigateur et vérifiez le résultat !

5️⃣ Points à retenir

  • 📌 **`<table>`** représente le tableau principal.
  • 📌 **`<tr>`** représente une ligne.
  • 📌 **`<th>`** est utilisé pour les titres de colonne.
  • 📌 **`<td>`** représente les cellules de données.
  • 📌 **`<thead>`**, **`<tbody>`**, et **`<tfoot>`** permettent d'organiser le contenu du tableau.
